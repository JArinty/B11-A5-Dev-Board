Question-1: What are the different ways to select an element in the DOM?
Answer:

Question-2: What is the difference between innerHTML, innerText, and textContent?
Answer:

innerHtml :It is used to get set of HTML content. It access the Html tag with content. It can access style ,script,HTML tag with content.
innerText : It used to get text content of element.It dose not access the style and script.It returns the text content visible to the user.
textContentIt is used to get HTML tags includes all text, even from hidden elements.It Returns all the text content inside the element, including hidden text .

Question-3: What is event delegation in the DOM?
Answer:

Event delegation is a technique use in event handaling.It add a single event listener to the parent element, and it captures events from its child elements.It optimized the code.

Question-4: What is event bubbling in the DOM?
Answer:

Event Bubbling is DOM Event Propagation where an event triggered on a child element propagates upward through its ancestors in the DOM.

Question-5: How do you create, add, and remove elements using JavaScript?
Answer:

    Create, Add, and Remove Elements using JavaScript

  • Create an Element:
    const new = document.createElement("div");
    new.textContent = "Create";
  • Add to the DOM:
    const new = document.createElement("div");
    document.body.appendChild(new);
  • Remove an Element:
    const new = document.createElement("div");
    new.remove();
